# 💻 معرض أعمالي المطور | الموقع الشخصي

موقع ويب شخصي (Portfolio) عصري ومحترف، مصمم بنظام الثيم الداكن (Dark Theme) ليعرض مهاراتي التقنية، آخر مشاريعي البرمجية، وقنوات التواصل الخاصة بي. تم بناء الموقع بالكامل ليكون متجاوباً وسلساً مع كافة الشاشات.

---

## ✨ المميزات (Features)

* **ثيم داكن فاخر (Modern Dark Theme):** ألوان مريحة للعين ومتناسقة للغاية (الخلفية الداكنة `#0f172a` مع لمسات الأزرق السماوي الجذاب `#38bdf8`).
* **تجربة تنقل ناعمة (Smooth Scroll):** خاصية الانتقال السلس التلقائي بين الأقسام عند الضغط على أزرار الهيدر.
* **تصميم مرن ومتجاوب (Fully Responsive):** بفضل استخدام **CSS Grid** وتوزيع الـ `auto-fit`، تترتب كروت المهارات والمشاريع تلقائياً لتناسب شاشات الموبايل، التابلت، والكمبيوتر بشكل مثالي.
* **تأثيرات حركية تفاعلية (Hover Effects):** ترتفع الكروت وتتغير ألوان الحدود والأزرار بسلاسة شديدة عند مرور الماوس لمنح المستخدم تجربة تفاعلية ممتعة.

---

## 📂 أقسام الموقع (Sections)

1. **الهيدر (Navbar):** يحتوي على الشعار وقائمة للتنقل السريع بين أقسام الموقع (الرئيسية، مهاراتي، أعمالي، تواصل معي).
2. **القسم الترحيبي (Hero Section):** واجهة ترحيبية قوية ومباشرة بعبارة "أهلاً بك، أنا مطور واجهات مستخدم" مع زر للذهاب إلى المعرض.
3. **المهارات التقنية (Skills):** شبكة ذكية تعرض مهاراتي الأساسية في التطوير وصناعة المحتوى (HTML5, CSS3, تجاوب الشاشات، صناعة المحتوى الرقمي).
4. **آخر المشاريع (Portfolio):** يعرض نماذج حية للمشاريع التي قمت بتطويرها (معرض سيارات رقمي، متجر زهور أونلاين، موقع مقهى ويب) مع إظهار تقنيات كل مشروع.
5. **التواصل (Contact & Footer):** أزرار تفاعلية مخصصة للمراسلة عبر الإيميل وزر مخصص باللون الأحمر الجذاب للوصول إلى قناتي على يوتيوب.

---

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **HTML5:** لبناء الهيكل البرمجي المنسق والمتوافق مع معايير السيو (SEO).
* **CSS3:** لتصميم الـ Layout باستخدام **CSS Grid & Flexbox**، والتحكم في حركات وتفاعل العناصر.
* **Google Fonts:** استخدام خط **Cairo** المتميز للأوزان المختلفة لضمان مظهر عربي أنيق.

---

## 🚀 طريقة التشغيل (How to Run)

1. تأكد أن ملف الـ HTML باسم `youssef.html` وملف الـ CSS باسم `youssef.css` متواجدين في مجلد واحد.
2. افتح ملف `youssef.html` في متصفحك المفضل للاطلاع على الموقع وتجربته.
